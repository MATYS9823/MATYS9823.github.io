📂 Ajouter vos fichiers de projets

Structure des dossiers

Les fichiers de votre portfolio doivent être organisés comme suit :

projets/ ├── chatbot-rapport.pdf ├── chatbot-screenshots.html ✓ ├── infrastructure-rapport.pdf ├── infrastructure-screenshots.html ✓ ├── database-rapport.pdf ├── database-screenshots.html ✓ └── analyse-rapport.pdf

📄 Rapports PDF

Pour chaque projet, ajoutez un fichier PDF de rapport :

  • chatbot-rapport.pdf - Rapport complet du chatbot
  • infrastructure-rapport.pdf - Documentation installation Debian
  • database-rapport.pdf - Rapport base de données
  • analyse-rapport.pdf - Rapport analyse économique
💡 Conseil : Vous pouvez exporter vos documents Word/Calc en PDF et les placer dans le dossier projets/

📸 Captures d'écran

Les pages de captures d'écran sont déjà créées :

  • chatbot-screenshots.html ✓
  • infrastructure-screenshots.html ✓
  • database-screenshots.html ✓

Pour ajouter vos images, éditez chaque fichier HTML et remplacez les placeholders par vos images PNG/JPG.

🚀 Comment faire ?

  1. Récupérez vos rapports PDF (Word → PDF)
  2. Placez-les dans le dossier projets/
  3. Prenez des captures d'écran de vos projets (interface, code, diagrammes)
  4. Sauvegardez-les en PNG ou JPG dans le dossier projets/
  5. Éditez les fichiers *-screenshots.html pour ajouter les images
