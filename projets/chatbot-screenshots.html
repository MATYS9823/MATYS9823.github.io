Description du Projet

Création d'un chatbot capable de répondre à des questions de culture générale avec un focus sur la fiabilité et la performance. Le projet compare différentes approches algorithmiques pour optimiser la vitesse de réponse.

Objectifs

  • Concevoir un algorithme efficient pour un chatbot QA
  • Minimiser la latence de réponse
  • Comparer plusieurs approches algorithmiques
  • Garantir la fiabilité des réponses

Résultats & Compétences

Livrables :

  • Conception et tests d'algorithmes en Java
  • Application Java fonctionnelle
  • Documentation des optimisations

Compétences :

  • Réaliser un développement d'application
  • Optimiser des applications informatiques
  • Travail en groupe, rigueur, remise en question
