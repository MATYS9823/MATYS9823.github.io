Description du Projet

Création d'une base de données relationnelle complète nommée "Le Tourmentin" en SQL. Le projet couvre la modélisation conceptuelle, la traduction en schéma relationnel, et l'implémentation avec scripts SQL complets.

Objectifs

  • Concevoir un MCD (SEA) conforme aux règles de gestion
  • Traduire le MCD en MLD (SLR) selon les règles de traduction
  • Définir et documenter toutes les contraintes d'intégrité
  • Implémenter la base de données en SQL
  • Valider la base avec des tests complets

Résultats & Compétences

Livrables :

  • MCD (SEA) avec justification des choix
  • MLD (SLR) avec traduction détaillée
  • Documentation des contraintes d'intégrité
  • Scripts SQL : création, suppression, tests
  • Bilan global et résultats de validation

Compétences :

  • Gérer des données de l'information
  • Modélisation de bases de données
  • SQL avancé et contraintes d'intégrité
