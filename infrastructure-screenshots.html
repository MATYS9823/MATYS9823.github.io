Description du Projet

Configuration complète d'un poste informatique dédié au développement. Installation d'un système d'exploitation Debian avec tous les outils essentiels (JDK, Git, IDE) et création d'une documentation visuelle du processus.

Objectifs

  • Installer et configurer Debian 13 avec KDE/Plasma
  • Mettre en place l'environnement de développement (JDK, Git)
  • Installer IntelliJ IDEA selon plusieurs méthodes
  • Documenter la procédure complète via une carte mentale

Résultats & Compétences

Livrables :

  • OS Debian 13 avec KDE/Plasma configuré
  • JDK et Git installés via packages Debian
  • IntelliJ IDEA Community (3 méthodes d'installation)
  • Carte mentale de la procédure

Compétences :

  • Administrer des systèmes informatiques complexes
  • Configuration Linux et gestion de paquets
